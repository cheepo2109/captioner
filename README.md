# captioner
Creates dynamic captions based on speech recognition.

## dependencies
- node.js
- npm

## usage
```bash
$ git clone https://github.com/mjrdnk/captioner.git
$ cd captioner
$ npm install
$ npm start
```

Then open a supported browser at [localhost:3000](localhost:3000) and enjoy the caption:
